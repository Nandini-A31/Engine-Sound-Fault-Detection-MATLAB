# Engine-Sound-Fault-Detection-MATLAB
## Project Overview
This project detects engine faults using acoustic signal processing techniques in MATLAB.
## Features
- Audio signal preprocessing
- Feature extraction (ZCR, RMS, Spectral Centroid, MFCC)
- Rule-based fault classification
## Tools Used
- MATLAB
## Output
The system classifies engine condition as Normal or Faulty based on extracted features.
